### Repositório de Desenvolvimento em linguagem R.
![R](https://img.shields.io/badge/R-4886ff?style=for-the-badge&logo=R&logoColor=white) 

This repository was created to explore and enhance my skills in R, a programming language focused on statistical analysis and data visualization. 
It aims to discover how to use R to manipulate datasets, perform advanced statistical analyses, create impactful visualizations, and develop predictive models. 
From the initial data exploration to the implementation of complex statistical solutions, this repository was created to expand my knowledge in the language.

Este repositório foi criado para explorar e aprimorar minhas habilidades em R, uma linguagem de programação voltada para análise estatística e visualização de dados. 
E descobrir como utilizar o R para manipular conjuntos de dados, realizar análises estatísticas avançadas, criar visualizações impactantes e desenvolver modelos preditivos. 
Desde a exploração inicial de dados até a implementação de soluções estatísticas complexas, este repositório foi criado para expandir meus conhecimento na linguagem.
